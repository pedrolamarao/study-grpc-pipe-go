Estudo em gRPC sobre _named pipes_ do Windows em linguagem Go.

Gerar o protocolo:

[source,shell]
----
protoc --go_out=. --go_opt=default_api_level=API_OPAQUE --go_opt=paths=source_relative --go-grpc_out=. --go-grpc_opt=paths=source_relative protocol/protocol.proto
----
